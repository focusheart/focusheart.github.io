First of all, you should install some neccesory tools to build source

yum install rpm-build redhat-rpm-config unifdef

wget

rpi -i

rpmbuild -bp --target=uname -m kernel.spec 2> err.log | tee out.log

you will get an err.log and an out.log, have a look in the err.log. some dependencies must be install before going on

cat err.log

yum install patchutils xmoto asciidoc elfutils-libelf-devel elfutils-devel binutils-devel newt-devel audit-libs-devel bison flex hmaccalc
